WebDesigner Project

A deep dive into design thinking, strategy, and user experience.



    WebDesigner is a dynamic platform I developed to help users design websites with ease and creativity. It provides intuitive drag-and-drop tools, pre-built templates, and live previews that simplify the web design process.

Easy UI experience

    Drag-and-Drop Simplicity The interface allows users to drag and drop elements (like text boxes, images, banners, and buttons) directly onto the canvas, making it intuitive even for beginners with no coding skills.
    Layout and Categories The sidebar is neatly divided into Design Elements and Layout, helping the user quickly find the building blocks needed for their landing page.
    Instant Feedback As soon as elements are added, users can see live changes in the canvas area, giving a What You See Is What You Get experience.
    Beginner-Friendly No need for technical knowledge, the UI is built for anyone to create a landing page by simply arranging elements visually.
    Export button By pressing the "Export" button you create and download an HTML file dynamically in the browser.
